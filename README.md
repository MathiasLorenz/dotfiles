# Description

Dotfiles, yay!

## Requirements

You need `git` and `stow` installed

## Installation

Check out the repo in your `$HOME` directory

```sh
$ git clone git@github.com/MathiasLorenz/dotfiles.git
cd dotfiles
```

Then use GNU `stow` to create symlinks

```sh
$ stow .
```

and voila, the config has been applied!
